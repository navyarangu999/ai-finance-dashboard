# 💰 AI Finance Dashboard

An AI-powered personal finance analyzer built with React and Claude API. Upload or paste your bank statement and get instant spending analysis, category breakdowns, personalized savings tips, and flagged suspicious transactions.

---

## 🚀 Live Demo
https://ai-finance-dashboard-jd9z76asv-navyarangu999s-projects.vercel.app/

---

## ✨ Features

- 📂 Upload `.txt` or `.csv` bank statements or paste raw text
- 🤖 AI-powered analysis using Claude API (Anthropic)
- 📊 Spending breakdown by category with interactive pie chart
- 💡 Personalized savings insights generated by AI
- ⚠️ Automatic flagging of suspicious transactions
- 📱 Fully responsive mobile-first design

## ⚙️ Getting Started

### Prerequisites
- Node.js v18+
- Claude API key from [console.anthropic.com](https://console.anthropic.com)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/ai-finance-dashboard.git

# Navigate into the project
cd ai-finance-dashboard

# Install dependencies
npm install

# Create environment file
echo "VITE_CLAUDE_API_KEY=your-api-key-here" > .env

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🔑 Environment Variables

Create a `.env` file in the root of the project:

```

```

> ⚠️ Never commit your `.env` file. It is already added to `.gitignore`.

## 🚀 Deployment

This project is deployed on Vercel. To deploy your own:

```bash
npm install -g vercel
vercel
```

Add `VITE_CLAUDE_API_KEY` in Vercel → Project Settings → Environment Variables.

---

## 💡 How It Works

1. User uploads or pastes a bank statement
2. The text is sent to Claude API with a structured prompt
3. Claude returns a JSON object with categories, totals, and insights
4. The app renders the data as charts and insight cards
